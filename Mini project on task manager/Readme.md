# MINI TASK MANAGER PROJECT

1 -> server
     npm init -y 
     ( package.json install -> which is used to manage dependencies, scripts, and project metadata. It’s basically  the starting point for any Node.js project setup.)

     npm i express mongoose cors jsonwebtoken bcrytjs 
     - express → Minimal web framework for building APIs and handling routes.
     - mongoose → ODM (Object Data Modeling) library to interact with MongoDB using schemas and models.
     - cors → Middleware to enable cross-origin requests between frontend and backend.
     - jsonwebtoken → Library to generate and verify JWTs for authentication and authorization.
     - bcryptjs → Library to hash and compare passwords securely.
